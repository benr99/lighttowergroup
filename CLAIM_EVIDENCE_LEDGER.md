# Claim Evidence Ledger

This ledger identifies claims that require an owner-approved source before being used in public copy. “Internal record” does not mean independently verified.

| Claim / fact | Current source | Public status | Required action |
|---|---|---|---|
| $5M+ typical mandate size | Homepage FAQ / contact copy | Retained | Confirm that this is a current qualification threshold. |
| Principal-led engagement | Existing site copy | Retained | Confirm that Ben remains directly involved through funding for every engagement. |
| 250,000+ capital sources | Existing site copy | Removed from conversion copy | Define whether this is a contact database, firms, decision-makers, or active relationships; do not restore without a documented definition and methodology. |
| 80+ bridge lenders | Bridge page | Removed | Retain only with a dated internal coverage list and a definition of “active dialogue.” |
| 40+ life companies | Life-company page | Removed | Retain only with a dated internal coverage list and a definition of relationship / active lender. |
| No retainer / success-based compensation | Existing site copy | Removed from primary conversion hierarchy | Confirm with legal/compliance and document permitted, universally accurate fee language. |
| One-business-day response | Homepage contact copy | Removed | Restore only if operationally guaranteed and continuously monitored. |
| Rates, spreads, leverage ranges, and “lowest” claims | Service pages | Reduced where market-sensitive | Support with a dated source and review schedule before publishing as market guidance. |
| Six representative transactions | `transactions.json` | Used as the internal site source of truth | Confirm amount, type, asset, market, description, and close year for every item before external promotion. |
| Transaction details under NDA | Existing site copy | Retained | Define the approved NDA process and what can be disclosed. |

## Transaction consistency requirement

The homepage's static transaction cards were reconciled to `transactions.json` during this overhaul. Future changes must update the data file and any homepage / sales-material references in the same release. Do not create a separate, manually maintained set of tombstones.

## Compliance review flags

Before publication or paid promotion, review all public language on compensation, placement activity, investment advice, securities, lender relationships, transaction results, and product-specific financing terms with appropriate counsel or compliance personnel.
